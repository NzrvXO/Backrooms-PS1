## 📋 Инструкция по запуску

### Минимальная установка (15 минут)

1. **Создай Player** с PlayerController + PlayerLook
2. **Создай 5 комнат** (кубы с компонентом Room)
3. **Создай Fuse** (желтый куб + Fuse.cs)
4. **Создай Echo** (куб + Echo.cs + Animator)
5. **Создай Lift** (лифт + LiftManager.cs + Light + Audio)
6. **Создай GameManager** объект с компонентами
7. **Создай LevelGenerator** объект с компонентами
8. **Создай Canvas UI** с TextMeshPro элементами
9. **Установи все ссылки** в инспекторе

Детальная инструкция в **QUICKSTART.md** или **SETUP_GUIDE.md**

### Первый запуск

```
1. Нажми Play
2. Должно выдать: "Level generation complete!"
3. Видишь UI "FUSES: 0 / 3"
4. Можешь двигаться WASD
5. Камера вращается мышью
```

## 🎯 Основной игровой цикл

```
Старт
  ↓
Генерация процедурного уровня (50 комнат)
  ├─ Спавн 3 предохранителей (случайные комнаты)
  ├─ Спавн 1-3 Эхо (случайные позы)
  └─ Лифт в конце (закрыт, требует питание)
  ↓
Исследование уровня
  ├─ Поиск предохранителей
  ├─ Избежание Эхо (они наблюдают за тобой)
  └─ Случайные события (аномалии)
  ↓
Активация предохранителей
  ├─ При контакте < 2м: автоматическая активация
  ├─ UI обновляется: FUSES: 1 / 3
  └─ Предохранитель становится зелёным и исчезает
  ↓
После 3 предохранителей
  ├─ Мерцание света (2-3 сек)
  ├─ Звук генератора
  ├─ Звонок лифта
  └─ Открытие дверей лифта
  ↓
Финальный побег
  ├─ Найти лифт
  ├─ Войти в открытые двери
  └─ "ESCAPED!" на экране
  ↓
Конец уровня (R для перезагрузки, ESC для выхода)
```

## 🔧 Ключевые параметры для балансировки

**Движение:**
```
Walk Speed: 1.2 → медленнее для атмосферы VHS
Sprint Speed: 2.0 → спринт в 1.67x быстрее
```

**Камера (VHS):**
```
Camera Shake: 0.015 → микро-трясение
Movement Bob: 0.05 → покачивание при ходьбе
Look Inertia: 0.15 → задержка реакции взгляда
```

**Уровень:**
```
Room Count: 50 → количество комнат
Spawn Chance: 0.7 → разреженность комнат
Fuses Required: 3 → количество предохранителей
```

**Эхо:**
```
Observation Distance: 15m → дальность видения
Disappear Distance: 5m → дистанция исчезновения
Move Speed: 0.5 → скорость приближения
```

## ✨ Особенности MVP

1. **Простая, но эффективная механика**
   - Не нужен сложный AI
   - Эхо работают как простой автомат
   - Все события просто генерируются и работают

2. **Процедурная генерация**
   - Каждый запуск - новый уровень
   - Новые позиции предохранителей
   - Новые Эхо
   - Высокая переиграбельность

3. **Атмосфера через детали**
   - VHS эффекты на камере
   - PS1-стиль визуала
   - Медленное, инерционное движение
   - Механика наблюдения (зловещая долина)

4. **Быстро реализуемо**
   - Всё сделано за 1 сеанс
   - Готовые к использованию компоненты
   - Легко масштабируется

## 📞 Быстрая помощь

**Нет комнат?** 
→ Проверь LevelGenerator.roomPrefabs в инспекторе

**Эхо не движутся?**
→ Убедись что Echo имеет компонент Animator

**Предохранители не активируются?**
→ Проверь что Player имеет тег "Player"

**Лифт не открывается?**
→ Убедись что активированы все 3 предохранителя

**Нет UI?**
→ Проверь что TextMeshPro элементы установлены в UIManager
